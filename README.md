# Description

## Install

Clone this repository to your local machine
and run below:

```bash
$ npm install
```

## Support

-   Create React App in client folder.

-   To run the server side.

    ```bash
    $ npm run server
    ```

-   To run the client side.

    ```bash
    $ npm run client
    ```

-   To run both server/client side.

    ```bash
    $ npm run dev
    ```

-   To test.
    ```bash
    $ npm test
    ```
